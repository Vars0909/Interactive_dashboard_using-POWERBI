# Netflix Data Analysis Dashboard using Power BI

I had used tools like **Power BI, Power Query, DAX, and Jupyter Notebook** to perform data cleaning, transformation, analysis, and visualization for this project.

In this task, I created an interactive **Netflix Content Analysis Dashboard** to explore and understand patterns in Netflix’s movies and TV shows dataset. The project involved preparing the dataset, creating calculated measures using DAX, and designing an interactive dashboard with a Netflix-inspired user interface.

The dashboard provides key insights about Netflix’s content library through different visualizations:

* **KPI Cards** were created to display important metrics such as total number of titles, total movies, and total TV shows available in the dataset.
* A **Movie vs TV Show distribution chart** was created to compare the proportion of movies and TV shows, helping understand the content strategy of Netflix.
* A **content growth trend analysis** was created using release year data to observe how Netflix’s content library expanded over time. The analysis shows a significant increase in content production in recent years compared to earlier years.
* A **Top Countries analysis** was performed to identify countries contributing the highest number of Netflix titles, highlighting the major content-producing regions.
* A **Content Rating distribution chart** was created to analyze the audience categories and understand the popularity of different ratings available on Netflix.
* Interactive **slicers** were added for filtering content based on type and release year, allowing users to explore specific categories dynamically.

Key insights obtained from the dashboard:

* Netflix has a larger collection of movies compared to TV shows, showing a stronger focus on movie content.
* The number of titles added to Netflix increased significantly in recent years, indicating rapid expansion of the platform’s content library.
* A few countries contribute a major share of Netflix content, with the United States being one of the leading contributors.
* Content ratings such as TV-MA and TV-14 represent a significant portion of Netflix’s available titles, indicating a large amount of mature and teen-oriented content.

The final dashboard provides a user-friendly way to analyze Netflix’s content trends and demonstrates the use of business intelligence tools for data-driven decision making.
